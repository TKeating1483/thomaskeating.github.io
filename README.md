# Thomas Keating academic website

This is a static site ready for GitHub Pages.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `assets/thomas-keating.jpg`

## Publish with GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder.
3. In GitHub, open **Settings** -> **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)` folder.
6. Save.
7. Your site will appear at `https://YOUR-USERNAME.github.io/REPO-NAME/`.

## Use a custom domain
1. Buy the domain from a registrar.
2. In GitHub Pages settings, enter the domain under **Custom domain**.
3. In your registrar DNS, point the domain to GitHub Pages.
4. Enable HTTPS in GitHub Pages once DNS resolves.

## Quick edits
- Replace text directly in `index.html`.
- Update fonts, spacing, and colours in `styles.css`.
- Replace `assets/thomas-keating.jpg` if you want a higher-resolution portrait later.
